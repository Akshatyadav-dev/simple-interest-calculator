# Simple Interest Calculator

This repository contains a Bash script that calculates simple interest based on user input.  
It is part of a Git & GitHub final assignment where we practice GitHub UI, Git CLI, and contributions.

## Features
- Calculates simple interest using Principal, Rate, Time.
- Shell-based script (`simple-interest.sh`)
- Open-source with Apache License 2.0
- Includes Code of Conduct and Contributing guidelines

## Usage
Run the script using:

```sh
